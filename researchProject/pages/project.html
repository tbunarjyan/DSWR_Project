The Project

The outline of Project

  • Main Idea

    What is the project idea?

    The idea of the project is to obtain the real-time database of apartments in Yerevan that are for sale, to research and record the various feature affiliations and to use mathematical regression models to generate possible price interval for specific house features given.

  • Data and Methods for Obtaing it

    How we have obtained it?

    The database was obtained using data scraping, that is a technique in which a computer program extracts data from human-readable output coming from another program. We use real values from myrealty.am website to collect the "houseData" dataset.

  • Research Analysis and Regression Models

    What we have gained?

    Using the dataset "houseData" we were able to construct different dependency charts, including boxplotes, barcharts, histograms etc.

    More importanty, we were able to implement mathematical regression models in R to aquire a feasible price for human-imput features.

  • Web Application

    How we illustrate the resuts?

    To illustrate the result and implementation of the DSWR project, we have created the following web application that visually illustrates the visual accomplishments, we have achieved as a result of our project.
